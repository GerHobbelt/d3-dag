    Sets the sugiyama layout's size to the specified two-element array of numbers [ width, height ] and returns this {@link SugiyamaOperator}. When size is non-null the dag will be shrunk or expanded to fit in the size, keeping all distances proportional. If it's null, the nodeSize parameters will be respected as coordinate sizes.
